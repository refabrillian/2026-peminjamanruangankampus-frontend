# Changelog

Semua perubahan penting pada proyek **Peminjaman Ruangan Kampus - Frontend** akan didokumentasikan di file ini.
Format ini mengikuti standar [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) dan menggunakan [Semantic Versioning](https://semver.org/lang/id/).

## [1.0.0] - 2026-02-15

### Added
- Inisialisasi proyek menggunakan Vite + React + TypeScript.
- Integrasi API menggunakan Axios untuk mengambil data dari Backend.
- Dashboard daftar peminjaman ruangan.
- Form input peminjaman ruangan baru.
- File konfigurasi `.env.example` untuk URL API.

### Changed
- Update `README.md` dengan panduan instalasi `npm install` dan penggunaan.
